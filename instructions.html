Pattern Copy Game

Game Instructions

Game Goal

The goal of the game is to study a pattern shown on a grid, remember it, and then copy it correctly after the pattern disappears.

How to Play

  1. Enter your player name on the launcher page.
  2. Select the difficulty level.
  3. Select the grid size.
  4. Choose a theme such as Colours, Symbols, or Lights.
  5. Select optional features such as hints or strict mode.
  6. Click Open Game Window.
  7. In the game window, click Start Game.
  8. Study the highlighted pattern during the preview phase.
  9. When the preview ends, click the correct tiles to copy the pattern.
  10. Click Check Pattern to submit your answer.

Difficulty Levels

  • Easy: fewer tiles in the pattern and longer preview time
  • Medium: moderate pattern size and preview time
  • Hard: more tiles in the pattern and shorter preview time

Buttons in the Game Window

  • Start Game starts a new game
  • Check Pattern checks your copied pattern
  • Next Round starts the next pattern round
  • Save Session stores the current game state
  • Load Session restores a saved game
  • Reset Game clears the current game
  • Back to Settings closes the game window and returns to the launcher

Scoring

  • Correctly copied patterns increase your score
  • Later rounds are worth more points
  • Strict mode gives less forgiveness for mistakes
  • Lives may be lost for incorrect answers
